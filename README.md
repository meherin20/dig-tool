# DigiTools

A fully responsive React-based digital tools storefront designed from Figma, featuring product browsing, cart management, pricing, and checkout interaction with toast alerts.

## Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON product data

## Core Features
- Product and Cart toggle section with default `Products` view and empty-cart state.
- Dynamic cart system with add, remove, total count in navbar, and `Proceed to Checkout` cart reset.
